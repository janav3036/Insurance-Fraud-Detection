Model Overview

To identify potentially fraudulent insurance claims, a supervised machine learning model was trained using the engineered dataset. The algorithm selected for this project is XGBoost (Extreme Gradient Boosting), a powerful ensemble learning method widely used for structured and tabular datasets.

Why XGBoost?
  • Strong performance on tabular datasets
  • Captures complex non-linear relationships between features
  • Built-in regularization helps reduce overfitting
  • Handles imbalanced datasets effectively
  • Highly scalable and widely used in real-world ML systems
Handling Class Imbalance

Insurance fraud datasets are naturally imbalanced because fraudulent claims are far less common than legitimate ones. To address this issue, the model uses the scale_pos_weight parameter in XGBoost to adjust the importance of fraudulent observations during training.

Additionally, the classification threshold was adjusted to prioritize recall. In fraud detection systems, identifying fraudulent claims is often more important than minimizing false positives.

Model Performance

The model was evaluated using several performance metrics including accuracy, precision, recall, and ROC-AUC.

  • ROC-AUC Score: ~0.63
  • Fraud Recall: ~0.75+
  • Accuracy: ~0.48

In fraud detection, recall is often prioritized because failing to detect fraudulent claims can lead to significant financial losses.
